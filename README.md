# RP2350B-RM2-Module

A compact RP2350B-based wireless module integrating:

- Raspberry Pi RP2350B
- RM2 Wi-Fi/Bluetooth module
- QSPI flash
- QSPI PSRAM
- USB-C
- Onboard voltage-regulator, capable of being disabled for production
- Single-sided design for automated production
- Castellated edges for embedded integration
- SWD debug support

The board is designed as a production-capable embedded module with integrated development convenience.

## Features

- RP2350B MCU
- RM2 wireless module
- 3.3V operation
- QSPI flash
- QSPI PSRAM
- USB device support
- Castellated module form factor
- JLCPCB assembly friendly
- Open hardware

Early prototype development.
